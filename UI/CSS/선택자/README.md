# 선택자


### 기본선택자
 ```
전체 선택자 : * {속성} 모든 태그를 지정  
타입 선택자 : 태그 {속성} 지정한 태그 스타일  
클래스 선택자 : .클래스 {속성} 지정한 클래스 스타일  
아이디 선택자 : #아이디 {속성} 지정한 아이디 스타일 (유니크 해야함)  
속성 선택자 : [속성] {속성선언;} 지정한 속성 또는 속성값이 있는 태그에 스타일 
```
